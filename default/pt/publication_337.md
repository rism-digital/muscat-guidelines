#### **Media type (337 $a)
Required field.**

Select the type of media from the following list:

- **Printed medium**
- **Electronic resource** : such as a database, website, or online catalog of works
- **E-book**
- **Printed music**
- **Manuscript** : a handwritten document
- **Media combination**
- **CD-ROM**
- **Microfiche**
- **Microfilm**
