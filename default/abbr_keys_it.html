Modo maggiore: A, B, C, D, E, F, G, C|x, D|x, F|x, G|x, A|b, B|b, C|b, D|b, E|b, G|b          

Modo minore: a, b, c, d, e, f, g, a|x, d|x, f|x, g|x, a|b, b|b, c|b, d|b, e|b, g|b

 

Esempi:

  • A = la maggiore
  • B = si maggiore
  • b = si minore
  • f|x = fa diesis minore
  • B|b = si bemolle maggiore
