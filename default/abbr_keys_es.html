Mayor (modo): A, B, C, D, E, F, G, C|x, D|x, F|x, G|x, A|b, B|b, C|b, D|b, E|b, G|b          

menor (modo): a, b, c, d, e, f, g, a|x, d|x, f|x, g|x, a|b, b|b, c|b, d|b, e|b, g|b

 

Ejemplos:

  • A = La mayor
  • B = Si mayor
  • b = si menor 
  • f|x = fa sostenido menor 
  • B|b = Si bemol mayor
